# 010. Обязательная проверка Ruff не проходит

- **Серьёзность:** Low
- **Статус:** Resolved
- **Затронутые файлы:** `app/`

## Описание

`uv run ruff check .` завершился с 45 нарушениями: неотсортированные импорты, строки длиннее лимита, неиспользуемые импорты, повторное определение `get_resume` и другие правила конфигурации проекта.

## Влияние

Локальный quality gate неработоспособен, а часть диагностик Ruff скрывает полезные сигналы о новых нарушениях.

## Рекомендуемое решение

Исправить текущие нарушения и добавить запуск `ruff check .` в CI как обязательную проверку pull request.

## Решение

Нарушения Ruff устранены; локальная обязательная проверка выполняется командой
`uv run ruff check .`. Workflow `Backend quality` запускает Ruff вместе с
проверками форматирования, mypy и pytest.
