# 009. Автоматические тесты отсутствуют

- **Серьёзность:** Medium
- **Статус:** Resolved
- **Затронутые файлы:** `tests/`, `pyproject.toml:75-77`

## Описание

`pytest` собрал 0 тестов. Для аутентификации, CRUD резюме, healthcheck, Redis-ротации JWT и миграций нет автоматического покрытия.

## Влияние

Регрессии в критичных HTTP- и security-сценариях попадают в ветку без обнаружения; в частности, найденные в ревью ошибки не были пойманы тестами.

## Рекомендуемое решение

Создать unit- и integration-тесты в `tests/` для auth, resume, health и миграций. В CI считать нулевую коллекцию тестов ошибкой.
