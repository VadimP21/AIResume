# Purpose

Keep documentation synchronized with the codebase.

# Scope

Review and update when necessary:

- README.md
- AGENTS.md
- CLAUDE.md
- Architecture docs
- API documentation
- Deployment docs

# Checklist

Verify:

- setup instructions
- configuration
- environment variables
- endpoints
- architecture diagrams
- workflows
- examples

# Output Format

Updated documents

Summary of changes

Missing documentation

Recommendations

# Constraints

Update documentation only when implementation changes require it.

Do not invent undocumented functionality.
