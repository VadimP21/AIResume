# Git branches

Use this skill only for repository branch operations.

1. Read and follow `docs/workflows/git-branches.md` before any branch mutation.
2. Follow `AGENTS.md`; for commits, additionally follow
   `docs/workflows/commits.md`.
3. Do not include or alter unrelated changes or untracked files.
