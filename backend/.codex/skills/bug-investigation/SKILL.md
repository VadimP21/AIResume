# Purpose

Perform structured debugging.

# Workflow

1. Reproduce.
2. Locate failure.
3. Find root cause.
4. Explain.
5. Suggest fix.
6. Suggest regression test.

# Checklist

Identify:

- failing module
- failing dependency
- incorrect assumptions
- race conditions
- transaction issues
- async issues
